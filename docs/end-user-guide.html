Joanni Pharma End-User Guide

This document explains how to use all features of the Joanni Pharma web app for day-to-day operations.

Audience: End users | Scope: Dashboard, records, navigation, data import/export, reports, editing, and deletion workflows

1. Getting Started

  1. Open the app in your browser.
  2. You will land on the Dashboard by default.
  3. Use the menu icon in the upper-left corner to access all sections.
All data is saved in your browser storage. Use CSV export regularly for backup.

2. Main Menu and Navigation

  1. Click the menu icon in the header to open the Main Menu.
  2. Only one submenu can stay expanded at a time (accordion behavior).
  3. Expand Records to open Vendors or Items maintenance.
  4. Expand Data to Export/Import CSV.
  5. Expand Reports to export Dashboard data to Excel.
  6. Expand Guide to open this End User Guide page.
  7. Close the menu by clicking outside, pressing Esc, or toggling menu icon.

Figure: Main Menu sections and available actions.

3. Dashboard and Pull-out Date Filter

  1. Use the month filter at the top-left of the toolbar to narrow rows by pull-out date.
  2. Click Add Line to create a new inventory record.
  3. Review table columns: SKU, Description/Vendor, Qty, Expiry, Return Policy, Pull-out, Remarks, Comment, Actions.
  4. Rows may highlight based on status (for pull-out, expired, or completed conditions).

Figure: Dashboard toolbar and table layout.

4. Add Inventory Line

  1. Click Add Line on the Dashboard.
  2. In the Line Entry modal, select an item using SKU/description autocomplete.
  3. Enter Quantity (optional) and Expiry Date.
  4. Set Remarks and optional Comment.
  5. Click Save to add the row, or Cancel to discard.
Pull-out date is auto-calculated from Expiry Date and Item Return Policy.

Figure: Add Line modal with required fields.

5. Inline Editing in Dashboard

  1. Edit Remarks and Comment directly in each row.
  2. Comment fields wrap and auto-grow while typing for easier long-note editing.
  3. After editing, row action buttons change to Save and Cancel.
  4. Click Save to commit or Cancel to revert inline changes.
  5. Use Mark Complete to toggle completion status; row fill updates immediately.

Figure: Inline editing workflow in dashboard rows.

6. Vendor Maintenance

  1. Open Main Menu > Records > Vendors.
  2. Add or edit vendor name in the form.
  3. When editing a vendor, cursor focus moves to the Name field automatically.
  4. Use table action buttons to edit or delete an existing vendor.
  5. If a vendor is in use by items, deletion is blocked with a notification modal.

Figure: Vendor form and vendor table.

7. Item Maintenance

  1. Open Main Menu > Records > Items.
  2. Enter SKU, Description, and select Vendor via autocomplete.
  3. When editing an item, cursor focus moves to the SKU field automatically.
  4. Choose Return Policy:
    • By number of months
    • On the month of expiry
  5. Save item, then manage entries using table action buttons.

Figure: Item maintenance form and list.

8. Import and Export (CSV)

  1. Export CSV: Main Menu > Data > Export CSV.
  2. Import CSV: Main Menu > Data > Import CSV and choose file.
  3. Watch for modal notification after import (success or failure).
  4. For best results, import CSV files that were exported from this app.
Exported CSV includes completion status and leaves pull-out date blank so it is recalculated from expiry date and policy during use/import.

Figure: Data section actions for backup and restore.

9. Export Report (Excel)

  1. Open Main Menu > Reports > Export to Excel.
  2. The file contains Dashboard records and respects your current filter.
  3. The export includes a Completed column.
  4. Save the generated file for reporting or sharing.

10. Delete Confirmation and Notifications

  1. Deleting a vendor, item, or inventory row opens a confirmation modal.
  2. Click Delete to continue or Cancel to stop.
  3. Informational and error messages are shown in notification modals.

Figure: Confirmation modal used before deleting records.

11. Troubleshooting

Cannot delete vendor or item

This record is still referenced elsewhere. Remove dependent records first, then retry.

Import CSV failed

Use a CSV exported by this app and verify header/column order is unchanged.

No rows shown on dashboard

Clear or change the pull-out month filter and confirm inventory records exist.
